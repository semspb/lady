Добавить проект

Инструкция:

  1. Заполните название и описание
  2. Вставьте URL картинки ИЛИ выберите файл
  3. Нажмите "Добавить проект"
  4. Скопируйте появившийся JSON
  5. Перейдите в репозиторий GitHub → projects.json → Edit → вставьте новый JSON
  6. Загрузите картинку в папку /projects/images/ (если выбрали файл)
